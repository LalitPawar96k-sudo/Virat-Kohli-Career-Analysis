# Virat-Kohli-Career-Analysis
# 🏏 Virat Kohli Career Analysis Dashboard | Power BI

## 📌 Project Overview

The **Virat Kohli Career Analysis Dashboard** is an interactive Power BI project that provides a comprehensive analysis of Virat Kohli's international cricket career. The dashboard highlights his batting performance across different formats, opponents, venues, and years using dynamic visualizations and KPIs.

This project demonstrates how sports analytics can be leveraged to uncover player performance trends and support data-driven storytelling through Power BI.

---

## 🎯 Objectives

- Analyze Virat Kohli's international batting career.
- Compare performance across ODI, Test, and T20I formats.
- Identify top-performing years and opponents.
- Evaluate consistency using batting statistics.
- Build an interactive dashboard with dynamic filtering.

---

## 📊 Dashboard Features

- 🏏 Career Overview
- 📈 Runs Scored Analysis
- 💯 Total Centuries & Half-Centuries
- 🎯 Batting Average Analysis
- 🏆 Performance by Format (ODI, Test, T20I)
- 🌍 Runs Against Different Opponents
- 📅 Year-wise Performance Trend
- 🏟️ Venue-wise Performance
- ⭐ Highest Individual Scores
- 🎛️ Interactive Filters & Slicers

---

## 📈 Key Performance Indicators (KPIs)

- Total Matches
- Total Innings
- Total Runs
- Batting Average
- Strike Rate
- Highest Score

- Total Half-Centuries (50s)
